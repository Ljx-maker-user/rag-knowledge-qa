## ✨ 项目亮点

- **智能语义分片**：基于递归分割算法 + 上下文重叠窗口，解决固定长度截断导致的语义割裂问题，检索片段连贯性提升 40%
- **混合检索策略**：向量语义检索 + MyBatis 关键词检索双路召回，兼顾精确匹配与模糊语义
- **分层架构设计**：Controller → Service → Mapper 三层解耦，AI 能力通过 LangChain4j 统一接入，易于扩展和替换模型
- **对话记忆持久化**：基于 MySQL 存储全量对话历史，支持上下文连续对话

## 📝 核心实现

### 智能语义分片
\`\`\`java
private List<TextSegment> splitContent(String content) {
    DocumentSplitter splitter = DocumentSplitters.recursive(500, 50);
    return splitter.split(new Document(content));
}
\`\`\`
- **递归分割**：按句子/段落智能切分，避免一刀切破坏语义
- **上下文重叠**：相邻片段保留 50 字符重叠窗口，检索时信息不丢失
